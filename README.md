# Relief Kit Distribution – Starter Scaffold

This scaffold contains a minimal backend for the Relief Kit Distribution app.

### Quickstart
1. Start PostgreSQL  
   Run `docker compose up -d` or start the container via Docker Desktop.
2. Install dependencies  
   Open `backend` folder → `npm install`.
3. Copy `.env.example` to `.env` and adjust if needed.
4. Apply SQL schema (`sql/schema.sql`) using pgAdmin or psql.
5. Start server (`npm run dev`) and visit http://localhost:3000/health.
